.Java build and project management tool

.Manages dependencies and automates the build process

.Maven is a free and open-source build automation tool developed by the Apache software foundation

.It is primarly used to automate the build process of Java projects,including tasks such comiling code,managing dependencies,packaging,testing,and deployment.

What Maven can do?

.Create the standard project folder strucuture

.Download and manage required dependencies/libraries automatically

.Execute unit test cases using testing frameworks like JUnit

.Compile the project source code

.package the project into deployable formats such as JAR or WAR

examples:

0.0.1-SANPSHOT,1.0-RELEASE

What is a Build Tool?

A build tool is software that automate the process of building your project.

“Building” means:

Compiling source code(Java → bytecode).

Packaging code into JAR/WAR files.

Managing dependencies(external libraries like Hibernate,Spring,etc).

Running tests.

Deploying the application.

When you build your project,the output can be packaged in JAR(Java Archive)/WAR(Web Application)

Bundles your code plus all dependencies.
